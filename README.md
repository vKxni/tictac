# Tictac
Uhm, yea.

Open a terminal and run

`mix run -e TicTac.x` (to start as x)

or

`mix run -e TicTac.o` (to start as o)

### To run interactively

1. Start iex: `iex -S mix`
2. Run the app with `TicTac.x`, `TicTac.o` or any other functions in the app.
